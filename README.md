An interactive 16-bit pixel art map of Nepal, reimagined in the style of *The Legend of Zelda: A Link to the Past*.
> **"It's dangerous to go alone! Take this map."**
## About
This project renders the geography of Nepal as a classic SNES overworld map. 
- **Authentic Shape**: "Knife-like" diagonal landmass.
- **Landmark Transposition**: Everest as Death Mountain, Kathmandu as Hyrule Castle.
- **Interactive**: Features animated sprites using pure CSS.
##  Tools & Tech Stack
- **HTML5**: Semantic markup and structure.
- **CSS3**: Keyframe animations for sprite movement and lighting effects.
- **Pixel Art Workflow**: Custom 16-bit asset generation and sprite optimization.
